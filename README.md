# Qris-Dinamis
make qris statis to dinamis

## Fitur
- Mengatur tagihan qris sesuai nominal
- Fitur pajak (opsional)
- Output image qris

## Cara Pakai
ubah file ```qris.js``` sesuai data yang dibutuhkan
```
$ git clone https://github.com/razisek/Qris-Dinamis
$ cd Qris-Dinamis
$ npm install
$ node qris.js
```
